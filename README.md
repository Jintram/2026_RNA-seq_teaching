

## Description

Description of files:

- `tutorial-bulkseq-synthetic-data.qmd`
    - Will generate a fictional "count table" to illustrate RNA-seq analysis. Explanations in file.
    Data is exported to the `data/` directory. A previously generated dataset is stored in
    `data_frozen/`, and used for analysis by `tutorial-deseq2-fakedata.qmd`.
- `tutorial-deseq2-fakedata.qmd`
    - Performs a DESeq2 analysis to illustrate core RNA-seq / DESeq2 analysis principles.
    
    
## To generate html files


```{bash}
# In terminal, with quarto installed, use:
quarto render tutorial-generate-synthetic-data.qmd --output-dir html/ --no-clean
quarto render tutorial-deseq2-fakedata.qmd --output-dir html/ --no-clean
```

Pre-rendered .html files are now included in the `html/` folder.
